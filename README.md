## Why this repo

Three reasons:

1. **Retention** — Writing and revisiting notes is more effective than passively reading docs.
2. **Speed** — Faster than searching Stack Overflow for something I've already solved.
3. **Clarity** — Explains my own reasoning and past mistakes, not just what the docs say.
